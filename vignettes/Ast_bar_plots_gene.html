Visualizing AST by Carbapenem Resistance

2025-09-20

# Introduction

The create_ast_barplots_gene() function generates barplots to visualize antimicrobial susceptibility test (AST) patterns for a set of drugs. The plots are faceted by carbapenem resistance status, which is inferred from the presence of carbapenemase genes in the column bla_Carb_acquired.

Steps Performed by the Function

  1. Subset relevant AST columns from the input dataset.

  2. Detect the carbapenem resistance indicator from Bla_Carb_acquired column:

    • CARBA-R if Bla_Carb_acquired has a carbapenemase gene present
    • CARBA-S if Bla_Carb_acquired has no carbapenemase gene or -

  3. Generate a stacked bar plot showing the proportion of S, I, R for each antimicrobial.

  4. Facet the plot by carbapenem resistance (CARBA-R vs CARBA-S).

  5. Save the plot as a PNG file in the specified output directory.

Usage

#import GenepiK
library(GenepiK)

import_data("~/git_repos/GenepiK/data-raw/rapid_report_dummy_data.csv", "~/git_repos/")
#create plot
create_ast_barplots_gene(masterdata, "~/git_repos/GenepiK/test_output/")
