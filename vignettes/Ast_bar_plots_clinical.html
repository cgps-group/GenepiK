Visualizing AST by Carbapenem Resistance from Clinical Data

2025-09-20

# Introduction

The create_ast_barplots_gene() function generates barplots to visualize antimicrobial susceptibility test (AST) patterns for a set of drugs. The plots are faceted by carbapenem resistance status, which is inferred from the column Isolate_type based on the phenotypic data whether the isolates was submitted as carbapenem resistant or senetive.

Steps Performed by the Function

  1. Subset relevant AST columns from the input dataset.

  2. Detect the carbapenem resistance indicator from Isolate_type column:

    • CARBA-R or CARBA-S from column Ioslate_type

  3. Generate a stacked bar plot showing the proportion of S, I, R for each antimicrobial.

  4. Facet the plot by phenotypic carbapenem resistance (CARBA-R vs CARBA-S).

  5. Save the plot as a PNG file in the specified output directory.

Usage

#import GenepiK
library(GenepiK)
